# Interpreter In Python3

<h3>Basic Structure:</h3>

![Image of structure of Interpreter](https://github.com/toobazameer/Interpreter_in_Python3/blob/master/Images/Interpreter.png)

**Sample Code:**

  _program Main;
  var x, y, z : integer;
  begin { Main }
     y := 7;
     x := (y + 3) * 3;
     z := (x * 2);
  end.  { Main }_
